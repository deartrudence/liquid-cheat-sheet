Template Variables ‚ñ∫

Shopify objects that contain attributes that can be dynamically put on the page. For example, the product object contains an attribute called title that can be used to output the title of a product.

The handle is used to access the attributes of a Liquid object. By default, it is the object‚Äôs title in lowercase with any spaces and special characters replaced by hyphens (-). Every object in Liquid (product, collection, blog, menu) has a handle.

The paginate tag is built using the attributes of the paginate object. You can use the default_pagination filter as a quick and easy alternative.


{% assign collection = collections.frontpage %}
{% paginate collection.products by 8 %}
	

	  
Collection: {{ collection.title }}

	  {% for product in collection.products %}
	    {% include 'product-loop' %}
	  {% endfor %}
	



{% if paginate.pages > 1 %}


  
{{ paginate | default_pagination }}



{% endif %}
{% endpaginate %}

This code shows the variable collection being assigned to collections.frontpage. Then we use the paginate tag to start and declare that we want to paginate through the collection.products with 8 per page. Below this is where we display the products using the product loop snippet. Once that is done, we use the paginate object to check to see if there is more than one page. If so, we use the paginate filter to set default pagination navigation. We then end the whole section with the {% endpaginate %} tag
